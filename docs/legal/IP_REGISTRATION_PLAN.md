## Шаг 1: Подготовка документов

**Что нужно:**
- Паспорт гражданина Узбекистана
- Прописка (регистрация по месту жительства)

**Юридический адрес:**
Для ИП используется адрес прописки. Никаких дополнительных документов не требуется.

**Примечание:** Фото 3x4 нужны только при оффлайн-регистрации. При онлайн-регистрации через OneID — не требуются.

---

## Шаг 2: Выбор кодов ОКЭД

Для IT/EdTech платформы рекомендуются следующие коды:

| Код | Название | Назначение |
|-----|----------|------------|
| **62.01** | Разработка программного обеспечения | Основная деятельность |
| **62.09** | Прочие услуги в области ИТ | Техподдержка, консалтинг |
| **63.11** | Обработка данных | Хостинг, SaaS |
| **85.59** | Прочие виды образования | Онлайн-курсы |

**Важно:** Первый код считается основным видом деятельности. Лучше указать все 4 кода сразу — добавление новых кодов позже требует отдельной процедуры.

## Шаг 3: Онлайн-регистрация ИП

**Портал:** [new.birdarcha.uz](https://new.birdarcha.uz)

**Процесс:**
1. Войти через OneID (авторизация по паспорту)
2. Выбрать «Регистрация ИП»
3. Заполнить форму:
   - ФИО
   - Адрес (прописка)
   - Коды ОКЭД (выбираются из справочника)
   - Налоговый режим → **Оборотный налог 1%** (с 2026 года)
4. Оплатить госпошлину: **337,500 UZS** (онлайн через Click/Payme)
5. Подтвердить через SMS

**Срок:** 1-2 рабочих дня

**Результат:** Свидетельство о регистрации ИП в личном кабинете

## Шаг 4: Получение ЭЦП

**Электронная цифровая подпись нужна для:**
- Подписания налоговых отчётов
- Работы с Payme/Click (некоторые документы)
- Электронного документооборота

**Где получить:**

| Центр | Стоимость | Срок | Сайт |
|-------|-----------|------|------|
| E-IMZO | ~150,000 UZS | 1-2 дня | [id.e-imzo.uz](https://id.e-imzo.uz) |
| UNICON.UZ | ~200,000 UZS | 1-2 дня | [unicon.uz](https://unicon.uz) |

**Процесс:**
1. Подать заявку онлайн на сайте центра
2. Прийти в центр с паспортом для идентификации
3. Получить USB-ключ или мобильный ЭЦП

**Рекомендация:** Мобильный ЭЦП (M-IMZO) удобнее — не нужен USB-токен.

## Шаг 5: Открытие расчётного счёта

**Документы для открытия:**
- Паспорт
- Свидетельство о регистрации ИП
- ИНН (получите при регистрации ИП)
- Печать ИП (можно заказать за 50,000-100,000 UZS)

**Процесс:**
1. Записаться в отделение банка (или онлайн-заявка)
2. Прийти с документами
3. Подписать договор
4. Получить реквизиты счёта (обычно в тот же день)

---

## Шаг 6: Подключение Payme/Click

### Payme

1. Зайти на [merchant.payme.uz](https://merchant.payme.uz)
2. Нажать «Стать продавцом»
3. Заполнить анкету:
   - Данные ИП
   - Реквизиты банковского счёта
   - Описание бизнеса
   - URL сайта
4. Загрузить скан свидетельства ИП
5. Ждать одобрения: **5-10 рабочих дней**
6. Получить Merchant ID и Secret Key

### Click

1. Зайти на [my.click.uz](https://my.click.uz)
2. Создать аккаунт мерчанта
3. Добавить новый сервис
4. Указать данные ИП и банка
5. Ждать верификации: **5-10 рабочих дней**
6. Получить Service ID, Merchant ID, Secret Key

### Настройка в проекте

```bash
# server/.env
PAYME_MERCHANT_ID=your_merchant_id
PAYME_SECRET_KEY=your_secret_key

CLICK_SERVICE_ID=your_service_id
CLICK_MERCHANT_ID=your_merchant_id
CLICK_SECRET_KEY=your_secret_key
```

## Содержание и закрытие ИП

### Ежемесячные расходы при работе

| Статья | Сумма | Примечание |
|--------|-------|------------|
| Налог (1% от оборота) | Зависит от дохода | С 2026 года |
| Обслуживание счёта | 50,000-100,000 UZS | Зависит от банка |
| Социальный налог | 0 UZS | Если нет сотрудников |

### При нулевом доходе

- Налог = 0 (платишь только с оборота)
- Банк = ~50,000-100,000 UZS/месяц
- Отчётность = нулевая декларация (бесплатно, через ЭЦП)

**Итого при нулевом доходе:** ~50,000-100,000 UZS/месяц

### Закрытие ИП

| Параметр | Значение |
|----------|----------|
| Госпошлина | Бесплатно |
| Срок | 3-5 рабочих дней |

**Процесс закрытия:**
1. Погасить все долги по налогам
2. Закрыть расчётный счёт в банке
3. Подать заявление на birdarcha.uz или в налоговую
4. Сдать свидетельство ИП
